# ✨ 特性

* 🚪适合入门微服务的一个项目
* 📚内部服务使用`GRPC + Consul`通信，方便学习
* 😁使用`Gin + Gorm`开发，更易上手
* 🥽利用`Makefile`工程化管理，方便开发
* 🐉项目中使用到了`IoC、AOP、OOP、IOP、函数式编程`等编程思想
* 👀文档清晰、代码思路清晰
* ❄️利用雪花算法生成ID、MD5做快哈希提升速率、bcrypt做慢哈希提高安全性
* .....

# 🍻 使用环境

* go 1.19
* mysql 8.0
* mongodb 6.0
* consul 1.14.4
